# Crawler (asyncio)

## Python: educational asyncio crawler

[blog.codeasap.pl](https://blog.codeasap.pl/)

python3, asyncio, aiohttp, sqlite3, BeautifulSoup (bs4)


## Options

```text
$ ./crawler.py --help
```

## Run

```text
FQDN=mysite.lan URLS=http://mysite.lan make run 
```

## Tests

```
make test
```

Or

```
make test-pytest
```

## Coverage

```
make coverage
```
